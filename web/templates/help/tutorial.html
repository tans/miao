商家使用流程
第一步：注册账号
  1. 访问创意喵平台首页
  2. 点击"注册"按钮
  3. 选择"商家"角色
  4. 填写用户名、密码、手机号
  5. 完成注册并登录
第二步：充值余额
  1. 进入"我的钱包"
  2. 点击"充值"按钮
  3. 选择充值金额
  4. 选择支付方式（支付宝/微信/银行卡）
  5. 完成支付，余额实时到账
提示：发布任务时会冻结相应金额，建议充值足够的余额。
第三步：发布任务
  1. 进入商家工作台
  2. 点击"发布任务"按钮
  3. 填写任务信息：
    • 任务标题：简洁明了，如"设计公司Logo"
    • 任务描述：详细说明需求、风格、尺寸等
    • 任务类型：选择合适的分类
    • 单价：每份作品的报酬
    • 需求数量：需要多少份作品
    • 截止时间：任务结束日期
  4. 上传参考资料（可选）
  5. 提交任务，等待审核
注意：任务描述越详细，创作者越容易理解需求，作品质量也会更高。
第四步：审核投稿
  1. 任务发布后，创作者会认领并提交作品
  2. 进入"投稿管理"页面
  3. 查看创作者提交的作品
  4. 根据任务要求评估作品质量
  5. 选择操作：
    • 通过：作品符合要求，系统自动结算
    • 拒绝：作品不符合要求，填写拒绝理由
  6. 达到需求数量后，可以提前结束任务
重要：请在7天内完成验收，超时未处理的投稿将自动通过。
第五步：任务完成
  1. 所有投稿验收完成后，任务自动结束
  2. 冻结的余额自动解冻
  3. 可以在"任务管理"中查看历史任务
  4. 可以对创作者进行评价（可选）
商家最佳实践
如何提高作品质量？
  • 提供详细的任务描述和要求
  • 上传参考资料和示例
  • 设置合理的报酬和截止时间
  • 及时回复创作者的问题
如何避免纠纷？
  • 明确交付标准和验收条件
  • 公平公正地验收作品
  • 拒绝时提供具体理由
  • 保持良好的沟通
创作者使用流程
第一步：注册账号
  1. 访问创意喵平台首页
  2. 点击"注册"按钮
  3. 选择"创作者"角色
  4. 填写用户名、密码、手机号
  5. 完成注册并登录
第二步：浏览任务
  1. 进入"任务大厅"
  2. 浏览可用任务列表
  3. 使用筛选功能：
    • 按任务类型筛选
    • 按报酬范围筛选
    • 按截止时间排序
  4. 点击任务查看详情
  5. 仔细阅读任务要求和交付标准
第三步：认领任务
  1. 确认自己有能力完成任务
  2. 确认有足够时间在截止日期前完成
  3. 点击"认领任务"按钮
  4. 任务进入"我的认领"列表
  5. 开始创作
注意：频繁认领但不提交会影响信用等级，请谨慎认领。
第四步：提交作品
  1. 完成创作后，进入"我的认领"
  2. 找到对应任务，点击"提交作品"
  3. 填写作品说明
  4. 上传作品文件：
    • 支持图片（JPG、PNG、GIF）
    • 支持文档（PDF、DOC、DOCX）
    • 支持压缩包（ZIP、RAR）
    • 单个文件最大10MB
  5. 检查作品是否符合要求
  6. 点击"提交"按钮
  7. 等待商家验收
第五步：获得报酬
  1. 商家验收通过后，报酬自动转入钱包
  2. 平台收取10%服务费（优秀等级8%，卓越等级5%）
  3. 进入"我的钱包"查看余额
  4. 余额满10元可申请提现
  5. 选择提现方式（支付宝/微信/银行卡）
  6. 1-3个工作日到账
提示：完成任务可获得信用积分，提升等级可降低服务费。
创作者最佳实践
如何提高通过率？
  • 仔细阅读任务要求，确保理解需求
  • 按照要求的格式和规格提交作品
  • 保证作品质量，注意细节
  • 在截止时间前提交
  • 主动与商家沟通，确认需求
如何提升信用等级？
  • 按时完成认领的任务
  • 提高作品通过率
  • 避免被投诉
  • 积极参与平台活动
遇到问题怎么办？
  • 任务描述不清：联系商家确认
  • 作品被拒绝：查看拒绝理由，改进后重新提交
  • 验收不公：通过申诉系统维权
  • 其他问题：联系客服或查看帮助中心
