# 创意喵平台端到端测试

## 测试脚本说明

`browser_monkey_test.py` 是一个完整的端到端测试脚本，使用 agent-browser 库模拟真实用户操作，测试创意喵平台的所有核心业务流程。

## 测试覆盖范围

1. **用户注册**
   - 创作者账号注册
   - 商家账号注册

2. **用户登录**
   - 商家登录
   - 创作者登录
   - 退出登录

3. **商家功能**
   - 钱包充值
   - 发布任务
   - 查看投稿列表
   - 验收作品
   - 查看交易记录

4. **创作者功能**
   - 浏览任务大厅
   - 认领任务
   - 提交作品
   - 查看钱包余额

## 安装依赖

```bash
cd test
pip install -r requirements.txt
```

## 运行测试

确保服务器正在运行：

```bash
# 在项目根目录启动服务器
./miao-server
```

然后在另一个终端运行测试：

```bash
cd test
python3 browser_monkey_test.py
```

## 测试输出

测试脚本会生成以下输出：

1. **控制台输出**: 实时显示测试进度和结果
2. **截图**: 保存在 `test/screenshots/` 目录，记录每个关键步骤
3. **测试报告**: 保存在 `test/test_report_<timestamp>.txt`

## 测试数据

测试脚本会自动生成随机的测试数据：
- 用户名: `creator_<timestamp>` / `merchant_<timestamp>`
- 手机号: 随机生成的11位手机号
- 密码: `Test123456`

## 注意事项

1. 确保服务器运行在 `http://localhost:8080`
2. 测试会创建真实的数据库记录
3. 每次运行都会生成新的测试账号
4. 截图会占用磁盘空间，定期清理 `screenshots/` 目录

## 故障排查

如果测试失败：

1. 检查服务器是否正常运行
2. 查看 `screenshots/` 目录中的截图，定位失败步骤
3. 查看服务器日志 `server.log`
4. 检查测试报告文件中的详细信息

## 扩展测试

可以根据需要扩展测试脚本：

- 添加更多边界条件测试
- 测试错误处理流程
- 测试并发场景
- 添加性能测试
